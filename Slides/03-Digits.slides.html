Treinamento do modelo¶

O Scikit-learn fornece um conjunto de modelos, com uma interface com os seguintes métodos:¶

  • Presentes em todos estimators:
    • fit(): o mtodo para o qual passado o dataset. No caso dos supervised estimatores, possui dois parmetros, o primeiro sendo os features, o segundo sendo os labels/target.
  • Presentes nos supervised estimators:
    • predict(): aps treinado, retorna os labels para o conjunto de features passado.
    • predict_proba(): retorna a probabilidade para cada label, em modelos probabilsticos.
    • score(): retorna um valor de 0 a 1 representando quo boa a predio.
  • Presentes nos unsupervised estimators:
    • transform(): lembrar do PCA. Apenas transforma, usando os parmetros do modelo.
    • fit_transform(): lembrar do PCA. Ajusta os parmetros do modelo aos dados e os transforma.
